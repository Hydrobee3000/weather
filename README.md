# Installation <a name="Installation"></a>

- Clone the repository
   ```sh
   git clone https://github.com/Hydrobee3000/weather.git
   ```
- Install NPM packages
   ```sh
   npm install
   ```
- Run the project
   ```sh
   npm start
   ```
